class automatminer.featurization.tests.test_sets.TestAllFeaturizers(methodName='runTest')¶

Bases: unittest.case.TestCase

Class to ensure the featurizers available in featurizer files in matminer match exactly to those defined to AllFeaturizers class. This test is meant to catch events when a new featurizer is defined but not listed inside AllFeaturizers (e.g. by mistake).
